Large Scale Scientific Computation

This course will introduce the student to the fundamentals of parallel scientific computing. We will first introduce the important tools for scientific computing, such GNU gcc, gdb, Makefiles and version control systems. Then introduce the hardware of the high performance computing. This will include a study of CPU architectures, interconnects, and forms of parallel memory. The practicalities of programming these machines (MPI, OpenMP) will be introduced, a parallel Poisson solver will be provided to show how to solve a partial differential equation in parallel machine. Issues such as programming performance evaluation and optimization will be addressed. Then the parallel matrix algorithms will be studied. Finally, we will introduce the adaptive concepts in large scale scientific computing and the Parallel Hierarchical Grid(PHG) platform for adaptive finite element method. Prerequisites: Numerical Analysis, Numerical linear algebra, Numerical method of PDE, C or Fortran. Description of the Course: 16 Lectures, 3 hours per lecture.
