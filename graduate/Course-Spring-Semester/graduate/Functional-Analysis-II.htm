Functional Analysis - II

This is a course for graduate students and senior undergraduates. The students are suggested to learn basic functional analysis and real analysis in advance. The topics include:

Linear Functional Analysis

  1. Topological spaces
  2. Topological vector spaces
  3. Compactness
  4. Geometrical form of Hahn-Banach theorem
  5. Basic results on linear bounded operators

Spectral Theory

  1. Banach algebras and their spectral theory
  2. Spectrum of linear bounded operators
  3. Compact operators
  4. Fredholm operators

Linear Semi-groups

  1. Vector-valued functions
  2. Bochner integration and Pettis integration
  3. Linear semi-groups
  4. Representation of semi-groups
  5. Characterizations of infinitesimal generators
  6. Unitary semi-groups and Stone theorem
